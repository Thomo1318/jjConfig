# JJ Configuration Repository

Personal Jujutsu configuration with Synthwave84 theme and GitMCP integration.

## Quick Start

Clone and deploy:
```bash
git clone https://github.com/Thomo1318/jjConfig.git ~/dev/activeProjects/jjConfig
cd ~/dev/activeProjects/jjConfig
make deploy
```

## Features

- Synthwave84 colors
- 20+ aliases
- GitMCP integration
- jj-fzf TUI
